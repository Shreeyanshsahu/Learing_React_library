# React Learning UI

A React + Vite practice project that builds a YouTube-style learning dashboard with a responsive layout, category pills, sidebar sections, and a video grid.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React Icons
- ESLint

## Features

- Responsive navbar for desktop and mobile
- Horizontal course/category selector
- Sidebar with subscribed channels and quick-access actions
- Video card grid with thumbnail, channel, views, and likes
- Glassmorphism-inspired UI styling

## Project Structure

```text
src/
  Components/
    Navbar/
    Categories/
    AppContent/
  App.jsx
  main.jsx
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## Available Scripts

- `npm run dev` - Run the app in development mode
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint checks
